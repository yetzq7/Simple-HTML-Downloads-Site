# Requirements
---
## Express

**How to install**

Run this in a cmd prompt in the folder:

    npm init -y
    npm install express



# How to use
Make a folder named "files" inside the same folder all these files are in

place your files inside there and place their names and file type in index.html

**Example:**

    a href="/files/file1.filetype">Download 1</a>

You can add more files by simply copy and pasting below the next and renaming it

*Made this in 8 mins so its very very simple*
